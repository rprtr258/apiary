## Building from Source

If you prefer to build Apiary from source, see the [Building from Source](/guide/installation#building-from-source) section in the installation guide.

## System Requirements

### Linux
- GLIBC 2.31 or newer
- X11 or Wayland display server
- 64-bit x86 processor

### macOS
- macOS 11.0 (Big Sur) or newer
- Intel or Apple Silicon processor

### Windows
- Windows 10 or newer
- 64-bit x86 processor

## Installation Instructions

### Linux/macOS

1. Download the binary for your platform
2. Make it executable:
   ```bash
   chmod +x apiary-linux-amd64
   ```
3. Run it:
   ```bash
   ./apiary-linux-amd64
   ```

### Windows

1. Download the `.exe` file
2. Double-click to run
3. If Windows SmartScreen blocks it, click "More info" then "Run anyway"
